## Вариант задания №6
Приложение состоит из трёх страниц, каждая из которых реализует расчёт по заданной формуле с графическим интерфейсом.

### Страница 1
- **Формула:**  
  β = √(10·(∛x + x^(y+2))) · (arcsin²(z) – |x – y|)  
- **Входные данные:** переменные x, y, z (вещественные числа)  
- **Выходные данные:** вычисленное значение β  
- **Особенности:** проверка области определения (подкоренное выражение ≥ 0, |z| ≤ 1), обработка ошибок.

### Страница 2
- **Формула:** условное выражение в зависимости от произведения x·b и выбора f(x):  
  - если 0.5 < x·b < 10: g = e^(f(x) – |b|)  
  - если 0.1 < x·b < 0.5: g = √(|f(x)| + |b|)  
  - иначе: g = 2·f(x)²  
- **Выбор f(x):** sh(x), x², eˣ (радиокнопки)  
- **Входные данные:** x, b  
- **Выходные данные:** значение g  

### Страница 3
- **Формула:** y = 9·x⁴ + sin(57.2 + x)  
- **Входные данные:** начальное x, конечное x, шаг dx  
- **Выходные данные:**  
  - таблица значений x и y в многострочном текстовом поле  
  - график функции (библиотека LiveCharts.Wpf) с легендой и осями  
- **Дополнительно:** страница помещена в ScrollViewer для удобного просмотра при большом количестве точек.

## Используемые технологии
- Язык программирования: **C#**  
- Платформа: **.NET Framework 4.7.2**  
- Графический интерфейс: **WPF** (Windows Presentation Foundation)  
- Построение графиков: **LiveCharts.Wpf** (устанавливается через NuGet)  
- Система контроля версий: **Git + GitHub**  

## Архитектура приложения
- Главное окно (`MainWindow`) содержит кнопки перехода на страницы и кнопку выхода с подтверждением.  
- Каждая страница (`Page1`, `Page2`, `Page3`) реализована как отдельный `Page`.  
- Навигация осуществляется открытием нового окна с содержимым страницы, главное окно скрывается.  
- Все поля ввода снабжены всплывающими подсказками, выполнена проверка корректности ввода (числа, диапазоны).  
- На третьей странице для отображения графика используется элемент `CartesianChart` из LiveCharts.Wpf; добавлена возможность прокрутки при переполнении.
